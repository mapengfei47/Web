### 1.2 实现方式

- 通常我们实现网页布局的方式大概有如下几种

> 1. float：浮动方式
> 2. position：定位方式
> 3. flexbox方式
> 4. table方式
> 5. 网格grid



## <span id='2'>二.左右布局</span>

### 2.0 通用配置

~~~css
html *{
    margin:0px;
    padding: 0px;
}
.layout article div{
    min-height:100px;
}
~~~

### 2.1 float实现

~~~HTML
<!-- 浮动方式实现左右布局 -->
    <section class="layout float">
        <style>
            .layout.float .left{
                float: left;
                width:300px;
                background-color: salmon;
            }
            .layout.float .right{
                background-color: pink;
            }
        </style>
        <h1>浮动实现</h1>
        <article class="left-right">
            <div class="left"></div>
            <div class="right">
                浮动解决翻案
                浮动解决翻案
                浮动解决翻案
                浮动解决翻案
                浮动解决翻案
                浮动解决翻案
            </div>
        </article>
    </section>
~~~

### 2.2 绝对定位

~~~html
<!-- 定位方式实现左右布局 -->
    <section class="layout absolute">
            <style>
                .layout.absolute .left-right>div{
                    position: absolute;
                }
                .layout.absolute .left{
                    width:300px;
                    left:0px;
                    background-color: salmon;
                }
                .layout.absolute .right{
                    left:300px;
                    right:0px;
                    background-color:pink;
                }
            </style>
            <h1>定位实现</h1>
            <article class="left-right">
                <div class="left"></div>
                <div class="right">
                    定位解决方案
                    定位解决方案
                    定位解决方案
                    定位解决方案
                    定位解决方案
                    定位解决方案
                </div>
            </article>
        </section>
~~~
