# 二.选择器语法

## 1. 基本选择器

| 选择器 | 类型       | 功能描述                             |
| ------ | ---------- | ------------------------------------ |
| *      | 通配选择器 | 选择文档中所有的HTML元素             |
| E      | 元素选择器 | 选择指定元素类型的HTML元素           |
| #id    | ID选择器   | 选择指定id属性值为‘id’的元素         |
| .class | 类选择器   | 选择指定C老师是属性值为‘class’的属性 |
|        |            |                                      |

**示例：**

~~~HTML
<style>
    /* 将所有元素的外边距和内边距设置为0*/
    *{
        margin:0px;
        padding:0px;
    }
    /* 将所有p标签的内容的颜色设置为红色*/
    p{
        color:red;
    }
</style>
~~~
